## Bug Reports<br>

Please state the following:<br>
1) **Version number** of the software, browser and browser version, and operating system type and version.<br>
2) **Expected behaviour** of the software, and what it does instead.<br>
3) **Steps to reproduce** the error. If the reproduction steps are fairly simple,<br>
  paste the code in the issue. If reproducing the issue requires a large amount of code, create a gist,<br>
  a separate snippet of code hosted on GitHub. If it is a hugely complicated issue, create a new<br>
  public  GitHub repo with some automated tests that demonstrate the bug.<br><br>

**Make sure to provide the code in a copy-pasteable format.**<br><br>

## Feature requests<br>
Feature request submissions should follow a similar format to bug reports.<br><br>

## Questions<br>
If you have a question about any part of the software, please be detailed.
